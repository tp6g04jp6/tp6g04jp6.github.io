Instagram Graph API

帳戶種類

  1. Instagram Business Accounts
  2. Instagram Creator Accounts

帳戶功能

  1. 獲取媒體 (get media)
  2. 發布媒體 (publish media)
  3. 管理回覆留言評論
  4. 識別被 @mention 到的用戶
  5. 找出帶有 #(hashtag) 標記到的媒體
  6. get basic metadata about other Instagram Businesses and Creators (metada包含哪些?)
  7. get metrics about other Instagram Businesses and Creators (metrics包含哪些?)

通常功能

  1. 獲取和管理已發布的照片，視頻和故事
  2. 獲取有關其他Instagram Business用戶和創作者的基本數據
  3. 審核評論及其回复
  4. 測量媒體和個人資料的互動
  5. 發現雜亂無章的媒體
  6. 發現@提及
  7. 發布照片和影片（目前處於封閉Beta版）

建議

如果只是要取得基本個人資料或照片等可以只使用instagram Basic Display API

限制

  1. 該API無法訪問Intagram消費者帳戶（即非企業或非創作者Instagram帳戶）。 如果您要為消費者用戶構建應用程序，請改用Instagram基本顯示API。
  2. 內容發佈僅適用於Instagram業務用戶。
