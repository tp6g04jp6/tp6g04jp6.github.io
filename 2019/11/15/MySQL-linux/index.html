安裝 net-tools, 使用 netstat 指令

設置MysSQL允許遠端訪問

編輯 mysqld.cnf

查詢 MySQL 設定檔, 並找到真正的設定檔案

若系統使用MySQL, config 檔可能是mysqld.cnf
若系統使用MariaDB, config 檔可能是50-server.cnf

修改bind參數
原本可能:bind = 127.0.0.1
修改為: bind = 0.0.0.0

進入 MySQL 服務

輸入後看到 mysql> 代表成功進入MySQL client

執行授權命令

在 MySQL 環境下執行授權命令(授權給遠程任何電腦登錄資料庫)，輸入以下指令：

PRIVILEGES ON . 代表權縣, ‘root’代表現在的使用者, ‘%’代表IP位置, 並輸入密碼
