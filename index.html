Instagram Graph API

Instagram Graph API

帳戶種類

  1. Instagram Business Accounts
  2. Instagram Creator Accounts

帳戶功能

  1. 獲取媒體 (get media)
  2. 發布媒體 (publish media)
  3. 管理回覆留言評論
  4. 識別被 @mention 到的用戶
  5. 找出帶有 #(hashtag) 標記到的媒體
  6. get basic metadata about other Instagram Businesses and Creators (metada包含哪些?)
  7. get metrics about other Instagram Businesses and Creators (metrics包含哪些?)

通常功能

  1. 獲取和管理已發布的照片，視頻和故事
  2. 獲取有關其他Instagram Business用戶和創作者的基本數據
  3. 審核評論及其回复
  4. 測量媒體和個人資料的互動
  5. 發現雜亂無章的媒體
  6. 發現@提及
  7. 發布照片和影片（目前處於封閉Beta版）

建議

如果只是要取得基本個人資料或照片等可以只使用instagram Basic Display API

限制

  1. 該API無法訪問Intagram消費者帳戶（即非企業或非創作者Instagram帳戶）。 如果您要為消費者用戶構建應用程序，請改用Instagram基本顯示API。
  2. 內容發佈僅適用於Instagram業務用戶。

印度(India) : 北印三角

前言

一直想找個機會紀錄印度旅行的一些心得與體驗
不過文筆不好只能以流水帳與看照片說故事的方式記錄
一段瘋狂疲憊但回想起來又念念難忘的旅程
在各城市移動的時間與方式有以google mymap做了簡單的紀錄

  • 台北-新加坡-新德里-列城 | airline
  • 列城-新德里-瓦拉納西 | airline
  • 瓦拉納西-新德里 | airline
  • 新德里-Jodhpur | train | 夜車約10小時
  • Jodhpur-Jaipur | bus | 約4小時
  • Jaipur-Agra | taxi | 約6小時
  • Agra-新德里 | bus | 約5小時

拉達克 (Ladakh)

印度機場

準備在英迪拉·甘地國際機場過夜, 等待前往Ladakh Leh的班機
在印度, 到處都有武裝警衛, 尤其是人多的場所, 更不用說國際機場內
武裝警衛隨身攜帶手槍步槍, 雖然台灣警察在巡邏時也會隨身攜帶手槍
但是在公眾場所看到一把把的舊式、改裝步槍, 非常不習慣
這樣的特殊景象也只有在當兵時才見過

機場的出入管制非常的嚴格, 你必須證明你有購買機票且在六小時內的班機才能進入機場
所以通常會印出購票證明, 上面必須要班機編號以及護照的相關資訊, 武警才會放人進入機場
所以建議到印度旅遊時先將自己航班且帶有自己護照姓名的電子機票先列印下來

列城

Leh 的機場, 是軍事基地的一部分, 航管人員全部都是軍人, 此地不可拍照
雖說如此, 但Vic在下飛機時把護照遺留在飛機內!!!
不過航管請我們不用擔心, 馬上連絡機組人員幫忙尋找並且有找回來了
此刻忽然覺得列城人真好 T.T

Leh 的位置在整個印度的最北方, 以地區來說屬於印控克什米爾地區,
左臨巴基斯坦, 右臨中國屬於邊境處, 在加上中印邊界問題, 所以派守的軍隊非常多
相對於武裝警衛, 這裡有非常多的正規軍人, 有些更靠近邊界的地區, 都需要申請邊防證
但中國人是無法申請的, 因此此處的中國遊客非常少
台灣也是因為前人抗議的努力, 印度政府才了解台灣與中國的不同
台灣在前些年才能申請邊防證
由於我們沒有預訂行程, 所以我們先到homestay安置完行李後
才上街尋找當地旅行社
但我們忘記我們處於海拔3500公尺高, 溫度-5~5度C的環境……

牛最大…

即使我們倉促的逛了一大圈, 我們也沒找到適合的旅行社, 而且在這種環境下, 理論上我們走個五分鐘應該就會氣喘如牛, 但我們能繞完一圈, 只能說是因為腎上腺素的關係, 很多背包客都誠心建議在初次到高海拔地方時, 先讓自己睡個七八小時的覺, 讓身體適應環境, 否則容易引起高山症, 我們就是活生生的案例, 當回過神時, Vic已經頭痛噁心, 這時只能回Homestay好好休息.
隔天一起床, homestay的老闆非常熱情, 幫我們找了travel agent以及告訴我們哪邊可以用比較好的匯率換到Rupee, 訂下接下來幾天的行程之後, 放心的閒逛起來

Main Bazzar Road

Tibetan Kitchen, 這裡的餐點都是藏族的料理, Homestay老闆首推來這吃momo, Thunkpa
momo就像小籠包的口感, Thunkpa則是加了很多辛香料的湯麵,

印度本來就是個民族大熔爐, 加上地緣關係這裡非常多的藏族,
北北印的人們給我的印象都很親切, 即使這裡屬於觀光區, 但不會強制推銷, 誘騙觀光客,
讓我們在談每件事時不需要擔心受騙, 若說有天要再來印度旅行, 我想Leh會是我的首選
